📁 1. Створення кампанії та налаштування тригерів

📌 Крок 1.1: Створіть нову кампанію

У розділі Campaigns → Create Campaign → оберіть тип Trigger Campaign.

📌 Крок 1.2: Налаштуйте тригер

Додайте подію, яка буде запускати кампанію. Наприклад:

  • User Login — вхід користувача
  • Custom Event — будь-яка кастомна подія
  • Segment Entry — вхід у сегмент

📌 Крок 1.3: Створіть Flow з Update Property

У конструкторі флоу додайте блок Update Property:

  • Виберіть Property: core_tags (або core_public_tags / core_external_markers)
  • Оберіть дію: Add to array / Add value

📌 Крок 1.4: Вставте код функції

У полі Value вставте JavaScript код відповідної функції:

Лейбл тега Що робить Куди вставляти
Рівень 1 Створює базовий тег (сегмент + депозит) Перший Update Property у флоу
Рівень 2 Створює універсальний тег (title) Другий Update Property у флоу
Рівень 3 Створює груповий тег (test/control) Третій Update Property у флоу

💡 Важливо:

Блоки Update Property мають йти послідовно один за одним. Спочатку перший рівень, потім другий, потім третій.

⚙️ 2. Налаштування параметрів функцій (для копіювання)

📌 Як змінити параметри кампанії

У кожній функції є блок НАЛАШТУВАННЯ на початку. Ось що потрібно змінити для нової кампанії:

🔧 Змінні, які можна налаштовувати:

Змінна Де знаходиться Що робить Приклад
VAR_PROJECT Всі 3 функції Назва кампанії (унікальний ідентифікатор) 'previp_daily_bonuses'
segmentCodes Функція 1 Відповідність кодів сегментів назвам '1559': 'Iron'
avgDepRanges Функція 1 Діапазони середнього депозиту { min: 0, max: 30, suffix: 'avg_dep_30' }
exchangeRates Функція 1 Курси валют для конвертації в USD EUR: 1.1

📝 Як додати новий сегмент:

var segmentCodes = {
    '205': 'Petrovich',  // Новий сегмент
    '228': 'Ace',        // Існуючий
    '1559': 'Iron'       // Існуючий (найвищий пріоритет)
};

💡 Важливо: Чим нижче в списку — тим ВИЩИЙ пріоритет! Останній сегмент у списку має найвищий пріоритет.

📝 Як змінити діапазони депозитів:

var avgDepRanges = [
    { min: 0, max: 29.999999, suffix: 'avg_dep_30' },      // 0-30
    { min: 30, max: 50, suffix: 'avg_dep_30_50' },         // 30-50
    { min: 50.000001, max: Infinity, suffix: 'avg_dep_51+' } // 51+
];

📝 Як змінити курси валют:

var exchangeRates = {
    USD: 1,
    EUR: 1.1,    // 1 EUR = 1.1 USD
    GBP: 1.20,   // 1 GBP = 1.20 USD
    CAD: 0.7     // 1 CAD = 0.7 USD
};

⚠️ Увага!

Якщо ви змінюєте VAR_PROJECT — потрібно змінити його В ОДИНАКОВО у всіх трьох функціях! Інакше теги не будуть пов'язані між собою.

🗑️ 3. Видалення тегів (якщо потрібно прибрати користувача з тесту)

📌 Крок 3.1: Створіть окрему кампанію для видалення

Створіть нову Trigger Campaign з тригером, який спрацьовує для потрібних користувачів.

📌 Крок 3.2: Додайте блок Update Property

У флоу додайте блок Update Property:

  • Property: core_tags
  • Action: Remove from array

📌 Крок 3.3: Вкажіть теги для видалення

У полі Value вкажіть теги, які потрібно видалити:

ab_test_previp_daily_bonuses_strata_iron_avg_dep_30,
ab_test_previp_daily_bonuses_title,
ab_test_previp_daily_bonuses_group_test

💡 Можна вказати кілька тегів через кому.

📌 Крок 3.4: Альтернативний спосіб (видалення за маскою)

Якщо потрібно видалити всі теги кампанії, використайте функцію:

(function() {
    var VAR_PROJECT = 'previp_daily_bonuses';
    var TAG_FIELDS = ['core_tags', 'core_public_tags', 'core_external_markers'];
    var tagsToRemove = [];
    
    for (var f = 0; f < TAG_FIELDS.length; f++) {
        var field = state[TAG_FIELDS[f]];
        if (!field) continue;
        
        var tags = Array.isArray(field) ? field : String(field).split(',');
        for (var i = 0; i < tags.length; i++) {
            var tag = tags[i].trim();
            if (tag.toLowerCase().indexOf(VAR_PROJECT.toLowerCase()) !== -1) {
                tagsToRemove.push(tag);
            }
        }
    }
    
    return tagsToRemove.join(',');
})();

Ця функція знайде та поверне всі теги, що містять назву кампанії, для видалення.

⚠️ Увага!

Після видалення тегів користувач знову зможе потрапити в тест при наступному запуску кампанії.

📄 4. Повні коди функцій (копіювати в Update Property)

🔵 Функція 1: Створення тегу першого рівня

(function() {
    // ========== НАЛАШТУВАННЯ ==========
    var VAR_PROJECT = 'previp_daily_bonuses';
    var STATIC_PREFIX = 'ab_test';
    var STATIC_MIDDLE = 'strata';
    
    var exchangeRates = {
        USD: 1, CAD: 0.7, AUD: 0.7, EUR: 1.1, GBP: 1.20, NOK: 0.1
    };
    
    var segmentCodes = {
        '205': 'Petrovich',
        '228': 'Ace',
        '762': 'Ace2',
        '750': 'Ace3',
        '256': 'Star',
        '1559': 'Iron'
    };
    
    var avgDepRanges = [
        { min: 0, max: 29.999999, suffix: 'avg_dep_30' },
        { min: 30, max: 50, suffix: 'avg_dep_30_50' },
        { min: 50.000001, max: Infinity, suffix: 'avg_dep_51+' }
    ];
    // ===================================
    
    var resultTag;
    var TAG_FIELDS = ['core_tags', 'core_public_tags', 'core_external_markers'];
    var hasCampaignTag = false;
    
    for (var f = 0; f < TAG_FIELDS.length; f++) {
        if (hasCampaignTag) break;
        var field = state[TAG_FIELDS[f]];
        if (!field) continue;
        var tags = Array.isArray(field) ? field : String(field).split(',');
        for (var i = 0; i < tags.length; i++) {
            if (tags[i].trim().toLowerCase().indexOf(VAR_PROJECT.toLowerCase()) !== -1) {
                hasCampaignTag = true;
                break;
            }
        }
    }
    
    if (hasCampaignTag) return resultTag;
    
    var convertedAmount = 0;
    if (state.acc_total_deposit_amount && state.acc_total_deposit_count) {
        var totalAmount = state.acc_total_deposit_amount || 0;
        var totalCount = state.acc_total_deposit_count || 0;
        if (totalCount !== 0 && totalAmount !== 0) {
            var avgRaw = totalAmount / totalCount;
            var currency = state.core_wallet_currency || '';
            var rate = exchangeRates[currency.toUpperCase()] || 1;
            var avgUSD = avgRaw * rate;
            if (!isNaN(avgUSD) && isFinite(avgUSD)) {
                convertedAmount = Math.floor(avgUSD * 10000) / 10000;
            }
        }
    }
    
    var segmentName;
    var externalSegment = state.core_external_segment;
    if (externalSegment) {
        var codes = String(externalSegment).split(',');
        for (var i = 0; i < codes.length; i++) codes[i] = codes[i].trim();
        var priorityOrder = [];
        for (var key in segmentCodes) priorityOrder.push(key);
        for (var p = priorityOrder.length - 1; p >= 0; p--) {
            var priorityCode = priorityOrder[p];
            for (var c = 0; c < codes.length; c++) {
                if (codes[c] === priorityCode) {
                    segmentName = segmentCodes[priorityCode];
                    break;
                }
            }
            if (segmentName) break;
        }
    }
    
    var avgSuffix;
    if (segmentName) {
        for (var r = 0; r < avgDepRanges.length; r++) {
            var range = avgDepRanges[r];
            if (convertedAmount >= range.min && convertedAmount <= range.max) {
                avgSuffix = range.suffix;
                break;
            }
        }
    }
    
    if (segmentName && avgSuffix) {
        resultTag = STATIC_PREFIX + '_' + VAR_PROJECT + '_' + STATIC_MIDDLE + '_' + segmentName.toLowerCase() + '_' + avgSuffix;
    }
    
    return resultTag;
})();

🟢 Функція 2: Створення універсального тегу (title)

(function() {
    // ========== НАЛАШТУВАННЯ ==========
    var VAR_PROJECT = 'previp_daily_bonuses';
    var STATIC_PREFIX = 'ab_test';
    var STATIC_TITLE = 'title';
    var UNIVERSAL_TAG = STATIC_PREFIX + '_' + STATIC_TITLE + '_' + VAR_PROJECT;
    var TAG_FIELDS = ['core_tags', 'core_public_tags', 'core_external_markers'];
    // ===================================
    
    var resultTag;
    var hasCampaignTag = false;
    
    for (var f = 0; f < TAG_FIELDS.length; f++) {
        if (hasCampaignTag) break;
        var field = state[TAG_FIELDS[f]];
        if (!field) continue;
        var tags = Array.isArray(field) ? field : String(field).split(',');
        for (var i = 0; i < tags.length; i++) {
            if (tags[i].trim().toLowerCase().indexOf(VAR_PROJECT.toLowerCase()) !== -1) {
                hasCampaignTag = true;
                break;
            }
        }
    }
    
    if (hasCampaignTag) {
        var hasUniversal = false;
        for (var f = 0; f < TAG_FIELDS.length; f++) {
            if (hasUniversal) break;
            var field = state[TAG_FIELDS[f]];
            if (!field) continue;
            var tags = Array.isArray(field) ? field : String(field).split(',');
            for (var i = 0; i < tags.length; i++) {
                if (tags[i].trim().toLowerCase() === UNIVERSAL_TAG.toLowerCase()) {
                    hasUniversal = true;
                    break;
                }
            }
        }
        if (!hasUniversal) resultTag = UNIVERSAL_TAG;
    }
    
    return resultTag;
})();

🔴 Функція 3: Створення групового тегу (test/control)

(function() {
    // ========== НАЛАШТУВАННЯ ==========
    var VAR_PROJECT = 'previp_daily_bonuses';
    var STATIC_PREFIX = 'ab_test';
    var STATIC_MIDDLE = 'strata';
    var STATIC_GROUP = 'group';
    var SUFFIX_TEST = '_test';
    var SUFFIX_CONTROL = '_control';
    
    var SEARCH_PREFIX = STATIC_PREFIX + '_' + VAR_PROJECT + '_' + STATIC_MIDDLE + '_';
    var GROUP_PREFIX = STATIC_PREFIX + '_' + VAR_PROJECT + '_' + STATIC_GROUP;
    var FULL_TEST_TAG = GROUP_PREFIX + SUFFIX_TEST;
    var FULL_CONTROL_TAG = GROUP_PREFIX + SUFFIX_CONTROL;
    
    var TAG_FIELDS = ['core_tags', 'core_public_tags', 'core_external_markers'];
    var HASH_PARAMS = ['user_id', 'user_ext_id', 'user_email', 'core_registration_date'];
    var HASH_SEPARATOR = '|';
    // ===================================
    
    var resultTag;
    var baseTag;
    
    for (var f = 0; f < TAG_FIELDS.length; f++) {
        if (baseTag) break;
        var field = state[TAG_FIELDS[f]];
        if (!field) continue;
        var tags = Array.isArray(field) ? field : String(field).split(',');
        for (var i = 0; i < tags.length; i++) {
            var tag = tags[i].trim();
            if (tag.toLowerCase().indexOf(SEARCH_PREFIX.toLowerCase()) === 0) {
                baseTag = tag;
                break;
            }
        }
    }
    
    if (baseTag) {
        var hasGroupTag = false;
        for (var f = 0; f < TAG_FIELDS.length; f++) {
            if (hasGroupTag) break;
            var field = state[TAG_FIELDS[f]];
            if (!field) continue;
            var tags = Array.isArray(field) ? field : String(field).split(',');
            for (var i = 0; i < tags.length; i++) {
                var tag = tags[i].trim().toLowerCase();
                if (tag === FULL_TEST_TAG.toLowerCase() || tag === FULL_CONTROL_TAG.toLowerCase()) {
                    hasGroupTag = true;
                    break;
                }
            }
        }
        
        if (!hasGroupTag) {
            var hashParts = [];
            for (var i = 0; i < HASH_PARAMS.length; i++) {
                hashParts.push(state[HASH_PARAMS[i]] || '');
            }
            hashParts.push(baseTag.toLowerCase());
            var hashString = hashParts.join(HASH_SEPARATOR);
            var hash = 0;
            for (var i = 0; i < hashString.length; i++) {
                hash = ((hash << 5) - hash) + hashString.charCodeAt(i);
                hash = hash & hash;
            }
            var groupSuffix = Math.abs(hash) % 2 === 0 ? SUFFIX_TEST : SUFFIX_CONTROL;
            resultTag = GROUP_PREFIX + groupSuffix;
        }
    }
    
    return resultTag;
})();
